# Velodesk Cockpit

Versão **local paralela** do Velodesk com cockpit operacional completo — copiloto inteligente em tempo real.

Roda isolada do app principal e do Lab (porta 8767), com `localStorage` próprio.

## Iniciar

```bat
start-cockpit.bat
```

Abra: **http://localhost:8768/?desk=v2**

## O que inclui

- Layout cockpit (1440px, grid 65/35)
- Painel de ação + próxima melhor ação
- Fila operacional compacta com priorização visual
- Central de eventos + timeline operacional
- Focus mode (Responder → atendimento sem distrações)
- Customer Intelligence Panel + Health Score
- Fila inteligente (SLA, churn, premium, recorrência)
- CRM Pipeline + Omnichannel timeline
- Automações operacionais (demo)
- Dashboard executivo (perfil Gestão)
- Sidebar moderna 200px recolhível

## Perfis

Use o seletor no header: Agente, Supervisor e Gestão.

### Supervisor — centro de comando

- Hero operacional com saúde (🟢/🟠/🔴) e previsão IA
- KPIs vivos + feed em tempo real
- Saúde operacional consolidada
- Distribuição de carga da equipe + leaderboard
- Riscos IA, gargalos, analytics, heatmap
- Visão por canal + previsão operacional
- Ações gerenciais rápidas
- **War Room Mode** automático em operação crítica
